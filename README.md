# Expense Tracker CLI

Una aplicación de línea de comandos para gestionar tus gastos personales de forma sencilla y rápida. Guarda todos los datos en un archivo JSON local, sin necesidad de base de datos.

URL consult: https://roadmap.sh/projects/expense-tracker

## Características

- ✅ Agregar gastos con descripción y monto
- 📋 Listar todos los gastos ordenados por fecha (más reciente primero)
- 🗑️ Eliminar gastos por ID
- 📊 Resumen total de gastos y del mes actual
- 💾 Almacenamiento local en archivo JSON
- 🎨 Interfaz en consola con colores y formato legible

## Requisitos

- Node.js (versión 14 o superior)
- npm (incluido con Node.js)
